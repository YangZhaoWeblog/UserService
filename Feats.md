# tiktok

# 核心关注点
+ 写一个日志库(可以放到自己的 Joe-pkg 里面的)
+ 多个微服务必不可少
+ 鉴定系统使用 jwt 自己写
+ 中间件做各种操作
 
+ 携程好好用
+ oss 系统用云端的
+ 从0到1自己实现

# 功能列表

## 视频功能
+ 视频推送(pull)
+ 视频投稿(push)
+ 视频列表(list)
 
## 用户功能
+ 用户登陆(jwt) 
+ 用户注册(jwt)
+ 用户(jwt)
 
## 点赞功能
+ 点赞
+ 取消点赞操作
+ 查看点赞视频列表
 
## 关注功能
+ 关注
+ 取关
+ 关注列表
+ 粉丝列表
 
## 评论功能
+ 发表评论
+ 删除评论
+ 查看评论
